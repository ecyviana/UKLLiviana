Soal MUdah 1 : Program di atas berfungsi untuk menghitung total biaya pengiriman barang berdasarkan berat, jarak, dan volume.
Cara kerjanya:
1. Pengguna memasukkan berat, jarak, panjang, lebar, dan tinggi barang.
2. Jika jarak ≤ 10 km → biaya = 4250 × berat,
   jika > 10 km → biaya = 6000 × berat.
3. Jika volume (panjang×lebar×tinggi) > 100 → tambah Rp50.000.
4. Hasil total biaya pengiriman ditampilkan ke layar.
